# Noah's Website

A clean, Apple-inspired personal website — built with React, Vite, and Framer Motion.

## Quick start

You need [Node.js](https://nodejs.org) 18+ installed (you have v24, so you're set).

```bash
# 1. Install dependencies (first time only)
npm install

# 2. Start the local dev server (opens http://localhost:5173 automatically)
npm run dev
```

That's it. The site reloads instantly as you edit files.

### Other commands

| Command           | What it does                                          |
| ----------------- | ----------------------------------------------------- |
| `npm run dev`     | Start the local development server with live reload.  |
| `npm run build`   | Create an optimized production build in `dist/`.      |
| `npm run preview` | Preview the production build locally.                 |

## Editing your content

**You almost never need to touch the code.** All text, projects, skills, and
links live in one file:

> **[`src/data/content.js`](src/data/content.js)**

Open it, change the values, save — the page updates. Every field is commented.
